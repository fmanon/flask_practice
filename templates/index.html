A very simple gateway into the world of cryptography is the Caesar Cipher.

It's a simple encryption process where each letter in a message is replaced by another letter some fixed number of positions away. For example, if we wanted to ecryption "Hi" with a shift of 2, we would take the "H" and move over 2 positions to the right to replace it with the letter "J". The same applies to the "i" - which is replaced by the letter "k". This would give us the encrypted message "Jk".
